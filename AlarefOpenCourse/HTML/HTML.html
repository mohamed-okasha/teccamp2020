Introduction to HTML5



What is HTML5

HTML stands for Hyper Text Markup LanguageHTML is the standard markup language for Web pagesHTML elements are the building blocks of HTML pagesHTML elements are represented by tags

Basic HTML Elements

    Tag Discription
    <!DOCTYPE html> The declaration defines this document to be HTML5
    <html> The element is the root element of an HTML page
    <head> The tag is used to contain specific information about a web page
    <title> The element specifies a title for the document
    <body> The text here element contains the visible page content
    <h1> The element defines a large heading
    <p> The element defines a paragraph
